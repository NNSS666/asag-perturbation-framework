# Coding scheme:
#   CATEGORY: article / domain_concept / verb / connective / other
#     - article: articles, determiners (the, a, an)
#     - domain_concept: technical Beetle vocabulary (terminal, battery, voltage,
#       circuit, current, switch, bulb, path, gap, closed, open, positive, negative,
#       connected, connection, polar markers)
#     - verb: main or auxiliary verb
#     - connective: prepositions, conjunctions (because, and, between, where, etc.)
#     - other: pronouns, modifying adverbs, deictics, quantifiers (both, each, always)
#       and generic nouns
#   SEVERITY: meaning_preserved / meaning_degraded / meaning_inverted
#     - preserved: a human grader would not penalise; meaning intact
#     - degraded: meaning less precise / partial loss, but inferable from context
