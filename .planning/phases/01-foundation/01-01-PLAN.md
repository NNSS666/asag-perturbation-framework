**`src/asag/schema/records.py` — Pydantic v2 models:**
- `QuestionRecord(BaseModel)`: question_id (str), prompt (str), rubric_text (Optional[str] = None), reference_answers (List[str]), score_scale (Literal["5way", "3way", "2way", "continuous"] = "5way"), corpus (Optional[str] = None). Use `model_config = ConfigDict(frozen=True)`.
- `AnswerRecord(BaseModel)`: answer_id (str), question_id (str), student_answer (str), gold_label (str), gold_score (float — normalized [0.0, 1.0]), annotator_id (Optional[str] = None). Use `model_config = ConfigDict(frozen=True)`.
- `PerturbationRecord(BaseModel)`: perturb_id (str), answer_id (str), family (Literal["invariance", "sensitivity", "gaming"]), type (str), generator (str), seed (int), text (str). Use `model_config = ConfigDict(frozen=True)`.

IMPORTANT: Use `from typing import Optional, List, Literal` (not `X | Y` syntax) — Python 3.9 compatibility required per CONTEXT.md.

**`src/asag/schema/grade.py` — Grade mapping:**
- Define `GradeLabel` as a string enum with values: correct, partially_correct_incomplete, contradictory, irrelevant, non_domain.
- Define `LABEL_TO_SCORE: Dict[str, float]` mapping: correct→1.0, partially_correct_incomplete→0.5, contradictory→0.0, irrelevant→0.0, non_domain→0.0.
- Define `LABEL_NAMES: List[str]` ordered by HuggingFace integer index: ["correct", "contradictory", "partially_correct_incomplete", "irrelevant", "non_domain"] (0→correct, 1→contradictory, 2→partially_correct_incomplete, 3→irrelevant, 4→non_domain).
- Add a helper function `label_int_to_score(label_int: int) -> Tuple[str, float]` that converts HF integer label to (label_name, gold_score).
- Add clear docstring documenting the mapping rationale: ASAG2024 benchmark confirms [0,1] normalization; partially_correct_incomplete at 0.5 is semantically half-credit; contradictory/irrelevant/non_domain are all incorrect variants.

**`src/asag/loaders/base.py` — Abstract base class (DATA-04):**
- Define `DatasetLoader(ABC)` with:
  - `@abstractmethod def load(self) -> Tuple[List[QuestionRecord], List[AnswerRecord]]` — returns (questions, answers) in canonical schema
  - `@property @abstractmethod def corpus_name(self) -> str` — human-readable name for run directory naming
- Import types from `asag.schema.records`
- Add docstring explaining that a new dataset = new subclass, no pipeline changes needed

**`src/asag/loaders/semeval2013.py` — SemEval 2013 loader (DATA-01):**
- `class SemEval2013Loader(DatasetLoader)` with constructor parameter `corpus: Literal["beetle", "scientsbank"]`
- `corpus_name` property returns f"semeval2013_{self.corpus}"
- `load()` method:
  1. Use `datasets.load_dataset("nkazi/Beetle")` or `datasets.load_dataset("nkazi/SciEntsBank")` based on `corpus` parameter
  2. **CRITICAL:** Concatenate ALL HuggingFace splits (train, test_ua, test_uq, test_ud) into a single list — do NOT use HF split names as Protocol A/B splits (see Pitfall 1 in RESEARCH.md). Some splits may not exist for all datasets; handle gracefully.
  3. Derive `question_id` by hashing question text: `f"{corpus}_{hashlib.md5(question_text.encode()).hexdigest()[:8]}"` (see Open Question 4 in RESEARCH.md). Log the question_id-to-text mapping.
  4. Deduplicate questions: group rows by question text, collect all distinct `reference_answer` values into a list, create exactly one `QuestionRecord` per unique question (see Pitfall 2 in RESEARCH.md)
  5. Create `AnswerRecord` for each row: use HF `id` field prefixed with corpus name as `answer_id` (e.g., "beetle_12345"), map integer label to gold_label string and gold_score float using `label_int_to_score()` from grade.py
  6. Set `annotator_id = None` for all SemEval rows (not available in HF mirror, see Open Question 3 in RESEARCH.md)
  7. Set `score_scale = "5way"` and `corpus = self.corpus` on all QuestionRecords
  8. Return (questions, answers) tuple
- Add logging: print corpus name, total questions loaded, total answers loaded, answers-per-question distribution (min, max, mean), label distribution
- Add `if __name__ == "__main__"` block that loads both Beetle and SciEntsBank and prints summary statistics
